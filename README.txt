# Design Studi â€” Streamlit Deployment Guide

1. Create a folder named Design_Studi
2. Copy these files into it:
   - app.py
   - preview_crawler.py
   - generate_sample_data.py
   - requirements.txt
   - README.txt
3. Create a free GitHub account if you don't have one
4. Create a new public repository, name it: design-studi
5. Upload all the files into the repository
6. Go to https://share.streamlit.io
7. Sign in with GitHub
8. Click "New app" -> select your repository -> branch "main"
9. Click "Deploy"
10. Your app will be live at a URL like:
    https://design-studi.streamlit.app
11. Use the checkbox in the sidebar to fetch live West Elm data or fallback demo data
12. Explore the dashboard, view color palettes, trending tags, and download CSV weekly digest
